Overview of Lab-1

In Lab 1 of the Web Application Programming and Hacking (WAPH) course with Dr. Phu Phung, participants explore the basics of the web and HTTP protocol. The lab includes tasks like understanding HTTP using tools like Wireshark and Telnet, and basic web programming in languages like C and PHP. It covers creating simple web applications, analyzing HTTP requests and responses, and understanding security risks. Overall, Lab 1 aims to provide hands-on experience with web technologies and security concepts.

Overview of Lab-2

Lab 2 revolves around front-end web development tasks. Participants engage in creating a basic HTML file featuring forms and implementing JavaScript functionalities like displaying the current date/time and dynamically manipulating content. Additionally, learners delve into advanced techniques such as utilizing Ajax requests, CSS styling, integrating the jQuery library, and incorporating Web API functionality. These tasks collectively aim to enhance participants' proficiency in front-end web development, equipping them with essential skills to create interactive and dynamic web applications.
